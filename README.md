# berkshares-contracts
EVM compatible Solidity contracts the Berkshares local currency program on the Celo protocol

## Install
```
nvm use 12
yarn --ignore-engines
``` 

## Build / Test / Deploy
`yarn build` / `yarn coverage` / `yarn migrate`
